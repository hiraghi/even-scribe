# Even Scribe

Even Realities G2 スマートグラス上で動く、日本語入力メモアプリです。
かな漢字変換でメモを書き、ノートはグラス内に保存します。
利用にはペアリングしたスマホの Even アプリが必要です。

## できること

- かな漢字変換による日本語入力（候補選択、文節の伸縮、変換学習）
- Shift＋英字で、日本語入力中でも英字（大文字）を優先候補として入力（例: `UI`・`AI`）
- ノートをグラス内に保存（オフラインで動作。変換候補の取得時のみ通信します）
- 最近のノート一覧・フォルダ表示・編集
- ノート／フォルダの新規作成・リネーム・削除（名前入力・削除確認はグラス画面上で完結）

## 動作環境

- Even Realities G2 とペアリングしたスマホの Even アプリ
- スマートフォンと接続された物理キーボード
- ネットワーク接続（かな漢字変換の候補取得に使用）

## 使い方

画面は「最近のノート一覧 → フォルダ表示 → 編集」の順に移動します。

### 一覧（最近のノート / フォルダ）

| キー | 動作 |
|---|---|
| `↑` / `↓` | 項目を移動 |
| `Enter` | 開く |
| `Esc` | 戻る（フォルダ表示では親フォルダへ） |
| `Ctrl+N` | 新規ノートを作成 |
| `Ctrl+Shift+N` | 新規フォルダを作成 |
| `F2` | 選択中のノート / フォルダをリネーム |
| `Del` / `Ctrl+D` | 選択中のノート / フォルダを削除 |

ブラウザ（スマホ）側の画面には **「New file」ボタン**もあり、押すと `Ctrl+N` と同じ名前入力ダイアログが開きます。

新規作成・リネームの**名前入力はグラス画面に表示**されます。日本語入力（かな漢字変換）で名前を入力し、`Enter` で確定、`Esc` で取り消します。リネームでは最初に現在の名前が選択された状態になり、そのまま打ち始めると名前全体を置き換えられます。名前を変えずに確定した場合は取り消しと同じ扱いです。

削除は**確認画面**で `Delete` / `Cancel` を選びます（矢印キーで選び `Enter` で決定、`Esc` で取り消し）。誤操作を防ぐため初期選択は `Cancel` です。フォルダを削除すると中のファイルもまとめて削除されます。

### 編集

| キー | 動作 |
|---|---|
| 文字キー | 入力 |
| `Enter` | 改行 |
| `↑` / `↓` | カーソルを移動（表示行単位） |
| `Home` / `End` | 行頭 / 行末へ |
| `PageUp` / `PageDown` | 画面単位でスクロール |
| `Shift` ＋ 上記 | 範囲選択 |
| `Backspace` | 1 文字削除 |
| `Ctrl+S` | 保存 |
| `Ctrl+Space` | かな入力の ON / OFF |
| `Esc` | 編集を終了（未保存の変更があれば保存 / 破棄を確認） |

グラス本体のタッチ／リングの上下スクロールは、編集画面では**1画面ずつのページ送り**になります（カーソルは動きません）。カーソルの移動は物理キーボードの矢印キー（`↑` / `↓`）で行います。かな漢字変換の候補が出ている間は、本体スクロールで候補を選べます。

### 日本語入力（変換中）

| キー | 動作 |
|---|---|
| `Space` | 変換 / 次候補 |
| `↑` / `↓` / `←` / `→` | 候補を選ぶ |
| `1`–`9` | 候補を番号で選ぶ（Classic のみ） |
| `Enter` | 確定 |
| `Shift`＋英字 | 英字（大文字）を優先候補として入力（例: `UI`・`AI`） |
| `Shift+←` / `Shift+→` | 変換範囲（文節）を伸縮 |
| `F10` | 変換せずに確定 |
| `Backspace` | 1 文字戻す |
| `Esc` | 変換を取り消す |

記号（`！` `？` `。` など）を入力すると変換中の文字列に取り込まれ、`Space` では**かな部分だけ**が変換されます（記号は末尾に残ります。例:「きょう！」→「今日！」）。変換範囲は `Shift+←` / `Shift+→` で調整できます。何も入力していない状態で記号を打つと、`！`/`!`/`‼`/`⁉` などの候補から選べます。

### 変換スタイル

かなから漢字への変換の出かたを 2 種類から選べます。スマホ側の設定フォームの
「IME conversion」で切り替えます（既定は Classic IME）。

- **Classic IME (Space to convert)** — 一般的な PC の IME と同じ方式です。かなを入力し終えてから `Space` で変換します。候補が並ぶので、`Space` / 矢印 / `1`–`9` で選んで `Enter` で確定します。
- **Live suggestions** — 入力しながら候補が自動で表示されます。`Space` を押さなくても候補が出るので、そのまま選んで確定できます。

## 開発

このリポジトリは npm workspaces 構成です（`client` と共有パッケージ `packages/jp-ime`・`packages/g2-core`）。
Node.js 20 以上が必要です。

```sh
npm install
npm --workspace client run dev     # 開発サーバ (http://localhost:5175)
npm --workspace client test
npm --workspace client run build
```

グラスが手元になくても、`start-emulator.cmd`（Windows）で Vite 開発サーバと EvenHub
シミュレータを起動し、動作を確認できます。
