# Changelog

エンドユーザー向け変更点まとめ。

形式は [Keep a Changelog](https://keepachangelog.com/ja/1.1.0/) に、バージョン番号は
[セマンティック バージョニング](https://semver.org/lang/ja/) に従います。

## [0.3.0] - 2026-07-14

### Added（追加）

- ノート・フォルダを削除できるようになりました。一覧で削除したい項目を選び、`Del` または `Ctrl+D` を押すと「Delete / Cancel」の確認画面が出ます。誤操作を防ぐため、初期選択は `Cancel` です。フォルダを削除すると中のノートもまとめて削除されます。

### Changed（変更）

- 編集画面のステータス表示を見やすく整理しました。IMEの状態（`[A]`/`[あ]`）・未保存マーク（`*`）・保存中の表示を画面**下部**（`Click` / `Double` のヒント行）に移し、変換候補のそばで確認できるようにしました。画面**上部**は行・列表示を `Ln 現在行/最大行, Col 現在列` に統一しました（グラス表示上の折り返し行を数えます）。
- 日本語入力中に記号（`！` `？` `。` など）を打っても、それまでのひらがなが即座に確定されなくなりました。記号は変換中の文字列に取り込まれ、変換すると「かな部分だけ」が変換されて記号は末尾に残ります（例:「きょう！」→ `今日！`）。変換する範囲は `Shift+←/→` で調整できます。前の文節を確定しても、未確定の記号はそのまま変換待ちで残ります。（何も入力していない状態で記号を打つと、従来どおり `！`/`!`/`‼`/`⁉` などの候補から選べます。）

### Fixed（修正）

- Shift＋子音キー（`LN` など）で入力した大文字英字が変換されず、スペースキーを押すと空白になって消えてしまう問題を修正しました。Shift入力の英字はそのまま確定できます。
- ローマ字入力の `du`・`di` が「どぅ・でぃ」になっていたのを「づ・ぢ」に修正しました（`だ ぢ づ で ど` の行）。あわせて `tcha`・`tchi`（`matcha` →「まっちゃ」）のような促音付きのつづりも変換できるようにしました。
- 別の画面でフォルダやノートを削除した後、その中のファイルが「最近のノート」に残ってしまい、開くと `File not found` エラーになる問題を修正しました。一覧に戻ると内容が最新の状態に更新されます。
- アプリを起動したまま保存したノートが「最近のノート」に反映されず、古い並びのままになっていた問題を修正しました。編集を終えて一覧に戻ると、保存したノートが先頭に表示されます。

## [0.2.0] - 2026-07-14

### Added（追加）

- フォルダを作成できるようになりました（`Ctrl+Shift+N`）。
- ノート・フォルダの名前を変更（リネーム）できるようになりました（`F2`）。
- 新規作成・フォルダ作成・リネームの**名前入力をグラス画面上で**行えるようになりました。スマホの画面を見なくても、これらの操作がグラスだけで完結します。名前も日本語入力（かな漢字変換）で入力できます。
- 保存せずに編集を終了しようとすると、「保存 / 破棄」を確認するようになりました。

### Changed（変更）

- `Ctrl+N` の新規ノート作成が、名前を入力するとすぐに一覧へ反映され、作成後は一覧に戻るようになりました。（以前は編集画面に入り、一覧を開き直すまで新しいノートが一覧に表示されませんでした。）
- Shift＋英字キーで入力した英字（大文字）が、日本語入力中でも最優先の変換候補になるようになりました（`UI`・`AI`・`OOM` など）。以前は「うい」などのかな候補が優先され、英字は候補の後ろのほうにしか出ませんでした。
- リネームを始めると現在の名前が選択された状態で表示され、そのまま打ち始めると名前全体を置き換えられるようになりました。

### Fixed（修正）

- 名前を変えずにリネームを確定すると、エラー（`Path already exists`）が表示されていた問題を修正しました。変更がない場合は取り消しと同じ扱いになります。
- ファイルを開いてカーソルを動かしただけでも、次回起動時に「未保存の下書き」の復元をうながされることがあった問題を修正しました。実際に編集した内容があるときだけ、下書きの復元を案内します。

## [0.1.0] - 2026-07-12

### Added（追加）

- 初回リリース。Even Realities G2 上で動く日本語入力メモアプリ。
- かな漢字変換による日本語入力（候補選択、文節の伸縮、変換学習）。
- ノートをグラス内に保存（オフラインで動作。変換候補の取得時のみ通信）。
- 最近のノート一覧・フォルダ表示・ノートの編集・新規ノート作成。
- 変換スタイルを Classic IME / Live suggestions から選択可能。
